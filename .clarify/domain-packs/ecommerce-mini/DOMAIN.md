# Domain: E-commerce (mini)

Online retail: browsing a catalog, building a cart, checking out, paying, and
fulfilling orders, with returns and refunds.

## Primary actors / roles
- Customer — browses, buys, returns.
- Admin / Merchant — manages catalog, prices, orders.
- Payment gateway — external charge/refund processor.
- Fulfillment / warehouse — ships orders.

## Core entities
- Product, Cart, Order, Payment, Shipment, Refund, Inventory.

## Typical goals
- Checkout, cart, promotions, order management, returns/refunds.

## When to select this pack
- The PRD mentions cart, checkout, products, orders, payments, or shipping.
