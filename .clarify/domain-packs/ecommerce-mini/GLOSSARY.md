# Glossary: E-commerce

| Term | Definition |
| --- | --- |
| SKU | Stock-keeping unit; a uniquely identifiable purchasable item variant. |
| Cart | A customer's pending selection of items before checkout. |
| Order | A confirmed purchase with line items, totals, and a lifecycle. |
| Capture | Finalizing a previously authorized payment. |
| Authorization | A hold on funds before capture. |
| Fulfillment | Picking, packing, and shipping an order. |
| Backorder | An order for an out-of-stock item to ship later. |
| Refund | Returning funds for all or part of an order. |
