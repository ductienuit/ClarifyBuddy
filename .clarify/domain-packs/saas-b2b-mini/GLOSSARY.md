# Glossary: B2B SaaS

| Term | Definition |
| --- | --- |
| Tenant / Org | An isolated customer account containing users and data. |
| Membership | The link between a user and an org, carrying a role. |
| Role | A named bundle of permissions (e.g. Admin, Member, Viewer). |
| Entitlement | A capability granted by the org's plan. |
| Seat | A billable user slot in a subscription. |
| Invitation | A pending request for a user to join an org. |
| SSO | Single sign-on via an external identity provider. |
