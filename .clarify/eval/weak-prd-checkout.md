# PRD: Checkout (draft)

We want to let people buy things on our site. The checkout should be fast and
easy to use.

## Features

- The user can check out.
- Payment is processed.
- The order is created.
- Inventory is updated.
- The user can manage their orders.
- Apply discounts when appropriate.
- Send a confirmation.

## Notes

- Refunds are allowed within 30 days.
- No refunds after 14 days.
- Also we should add a recommendation engine on the checkout page.
